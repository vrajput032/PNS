# PNS Enterprises

One-page website for **PNS Enterprises**, a Sonipat, Haryana-based manufacturer of MS (mild steel) pipes for geysers.

Plain HTML/CSS, no build step required.

## Run locally

Just open `index.html` in a browser, or serve it:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Structure

- `index.html` — page markup and content
- `style.css` — white / gold / navy theme styling
